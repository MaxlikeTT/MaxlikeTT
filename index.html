Project 1
Powershell Password Lock GUI

This project was coded in 93 lines of powershell using the powershell IDE. It's purpose is to provide security with a login much like the one aready used when you log into your computer. It blocks all windows commands, clicks, programs, etc, ensuring the only way to get passed is by the user typing the correct pre-defined user-pass. It was done on my own out of curiousty, and while it does not utilize the full power of powershell, it was still great to learn from.

Project 2
Hangman Python Game

This classic game of hangman was coded in about ~120 lines in Python using the pygame library. It has function buttons that adds or deletes images in the project, to make the game appear animated and smooth. Similar to the last project, this project was done on my own time, except this project was a follow along tutorial from a youtuber known as @techWithTim. I learned a lot from this project on how to make python functions work together to make a game or informative app, and the basic functions of the python langauge.
