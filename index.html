Project 1
Powershell Password Lock GUI

This project was coded in 93 lines of powershell using the powershell IDE. It's purpose is to provide security with a login much like the one aready used when you log into your computer. It blocks all windows commands, clicks, programs, etc, ensuring the only way to get passed is by the user typing the correct pre-defined user-pass. It was done on my own out of curiousty, and while it does not utilize the full power of powershell, it was still great to learn from.
